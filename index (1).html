Dokumentasi Project Stack Linked List

Deskripsi

Program ini merupakan implementasi struktur data Stack menggunakan Linked List.
Mendukung operasi dasar seperti:

  • Push
  • Pop
  • Display (Peek)

Cara Penggunaan

  1. Push
  2. Pop
  3. Display
  4. Exit

class Node

Representasi simpul tunggal dalam linked list.

Atribut

  • int data — Menyimpan nilai data
  • Node* next — Penunjuk ke simpul berikutnya

Konstruktor

  • Node() — Menginisialisasi next ke nullptr

class Stack

Kelas yang merepresentasikan struktur data Stack.

Atribut

  • Node* top — Penunjuk ke elemen paling atas

Konstruktor

  • Stack() — Menginisialisasi top ke nullptr

Metode

  • int push(int value) — Menambahkan elemen ke stack dan mengembalikan nilai yang dimasukkan
  • bool isEmpty() — Mengembalikan true jika stack kosong
  • void pop() — Menghapus elemen paling atas
  • void peek() — Menampilkan seluruh isi stack

Fungsi Global

  • int main() — Fungsi utama program (menyediakan menu interaktif)
